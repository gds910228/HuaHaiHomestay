# 画海民宿小程序 - 快速开始指南 🚀

## 方法一：从个人中心进入（推荐）✨

### 步骤：

1. **编译运行小程序**
   - 打开微信开发者工具
   - 点击「编译」按钮

2. **进入个人中心**
   - 点击底部Tab栏的「我的」
   - 滚动到底部

3. **点击「数据初始化」**
   - 在功能菜单中找到「数据初始化」
   - 点击进入初始化页面

4. **初始化数据**
   - 点击「📦 初始化全部数据」按钮
   - 等待执行完成（查看日志）
   - 数据初始化成功后点击返回

5. **查看效果**
   - 点击「首页」Tab查看攻略列表
   - 点击「民宿」Tab查看民宿信息

---

## 方法二：地址栏输入（快速）

### 步骤：

1. **找到地址栏**
   - 在模拟器上方有一个路径输入框
   - 当前显示为：`pages/home/index`

2. **输入新路径**
   - 删除当前路径
   - 输入：`pages/init/index`
   - 按回车键

3. **执行初始化**
   - 页面加载后点击「📦 初始化全部数据」
   - 等待执行完成

---

## 方法三：修改 app.json（临时）

### 步骤：

1. **打开 app.json**
   - 在微信开发者工具中打开
   - 找到 `pages` 数组

2. **调整顺序**
   ```json
   "pages": [
     "pages/init/index",    // 移到第一位
     "pages/home/index",
     "pages/guide-detail/index",
     ...
   ]
   ```

3. **保存编译**
   - 保存文件
   - 小程序会自动跳转到初始化页面
   - 初始化完成后改回原顺序

## 常见问题

**Q: 初始化失败怎么办？**
A: 检查云函数是否已部署（右键huahai和init-database文件夹）

**Q: 页面打不开？**
A: 确保云函数已部署，点击编译按钮重新编译

**Q: 数据库在哪里看？**
A: 点击开发者工具的「云开发」→「数据库」

**Q: 如何修改密码？**
A: 编辑 cloudfunctions/huahai/index.js 和 init-database/index.js
